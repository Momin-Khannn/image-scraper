Image Scraper Actions

A packaged Python scraper that runs in GitHub's cloud runners and publishes each scrape as a downloadable artifact.

Open workflow View repository
Cloud runner

The Python job runs on Ubuntu through GitHub Actions, so local machine uptime is not required.

Downloadable output

Each run uploads images and manifest metadata as an Actions artifact retained for 14 days.
